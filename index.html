📝 Write & Modify Script

Script files should be named in a file_name.extension format such as script.sh To create a file, use the touch command :

Using “touch” command
jvsb@ProDev:~$ touch script.sh

Alternative #1 : Use cat command to create a file with a redirect (>) :

Using “cat” command
jvsb@ProDev:~$ cat > script.sh
Enter the following while in “cat” command
#!/bin/bash
echo 'Hello World'

press CTRL + D while under cat to exit.

View script.sh with cat
jvsb@ProDev:~$ cat script.sh
#!/bin/bash
echo 'Hello World'

Alternative #2 : Use nano command to modify script :

Using nano
jvsb@ProDev:~$ nano script.sh
Content in nano
#!/bin/bash
echo 'Goodbye World'

press CTRL + O to Overwrite, CTRL + X to Exit.

Verify changes
jvsb@ProDev:~$ cat script.sh
#!/bin/bash
echo 'Goodbye World'

Permissions: r (read), w (write), x (execute).

Grant executable
jvsb@ProDev:~$ chmod +x script.sh
jvsb@ProDev:~$ ls -l script.sh
-rwx------ 1 jvsbPD jvsbPD 31 Aug 16 11:13 script.sh

* wildcard examples: *.sh matches all shell scripts.

Wildcard chmod
# all .sh files
jvsb@ProDev:~$ chmod +x *.sh
# everything in current dir
jvsb@ProDev:~$ chmod +x *

UGO & Octal examples

Symbolic method: u=rwx,g=rx,o=r etc.
